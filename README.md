# PhrasePal

**What is PhrasePal?**
An AI assistant that helps you learn idioms and phrases from videos.

Extracts tricky expressions from subtitles or video speech.

Explains meanings and gives examples to make learning easier.

Connects language learning to real-life conversations.


**Key Features:**
Upload video files or subtitle files (.srt, .vtt).

Transcribe video speech if subtitles aren’t available (using Whisper).

Automatically find idioms, phrasal verbs, and phrases using AI.

Show phrase meanings, usage examples, and timestamps.

Create quizzes or flashcards for practice.

User-friendly web interface to view phrases and test knowledge.


**How We’ll Build It:**
Start by parsing subtitle files to get video text.

Use Whisper to transcribe speech from videos without subtitles.

Use AI models (like GPT) to detect and explain idioms and phrases.

Build a simple web app with Streamlit or React for interaction.

Add quiz generation and learning progress tracking features.

Design the system so it can grow with more features later (voice, languages, etc.).
